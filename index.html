Why Llama Recipe Manager?

📦

Recipe-first workflow

Each saved recipe is a named bundle of llama-server flags. Switch models, contexts, and GPU layouts in one click.

🔒

Local-first

No account, no telemetry, no network dependency. Your recipes live in a SQLite database on your machine.

🛡️

Safety rails

Flag deny-list prevents conflicting arguments. Relative model paths are confined to your configured model directory.

🌍

Cross-platform

Native desktop experience on macOS, Linux, and Windows. Honours your system theme everywhere.

📊

Live metrics

Real-time GPU, memory, and throughput stats right in the app. No need to check external tools.

⚡

Auto-updates

Signed, silent background updates via GitHub Releases. Stay current without manual downloads.
